# MEME DASH

**MEME DASH** is a chaotic, hilarious, and ultra-addictive endless runner where you are a **Meme Lord** dashing through the viral internet streets.

## Features
- **Fast-paced gameplay**: Swipe to jump or tap to dash!
- **Meme Coins & Power-ups**: Collect trending memes to fill your hype meter and enter **Viral Mode**.
- **ERC-8021 Transaction Attribution**: Seamlessly integrated on Base Mainnet.
- **On-chain Score Submissions**: Prove your Meme Lord status with real SIWE signatures.
- **Trustless Agents (ERC-8004)**: On-chain deterministic behavioral agents interacting with the game economy.

## Setup & Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Agent Configuration
The application comes pre-configured with a `.well-known/agent-card.json` standard, enabling ERC-8004 capabilities and Model Context Protocol (MCP) interactions.
